Core4 Concept Technical Definition Real-Life Analogy Python Example Scratch Example
Variables a place to label and store data like a bucket: can store information, and what's stored can be changed or brought to where it's needed.
Loops a function that repeats a set of commands like a repeat sign in music: tells the computer to repeat the last few tasks it just completed.
Conditionals booleans that check to see if a condition is met, and act accordingly like binary: only two options, and based on the user, the computer switches between options.
Functions a set of commands combined into one command like switching from hand-written to typed: makes everything you've written easier to read.
